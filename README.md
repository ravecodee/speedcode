# @ravecodee - Speed Code

Speed code source code collection React Native support on Android & iOS

## Install

Open a Terminal in the project root and run:

```sh
yarn install
```
Open for android:

```sh
yarn android
```
Open for iOS:

```sh
yarn ios
```
